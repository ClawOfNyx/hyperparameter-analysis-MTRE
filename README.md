# Analysis of Hyperparameter Impact on MTRE Framework and Hallucination Detection

This repository contains code used and modified as part of the **Analysis of Algorithms to Reduce Hallucinations in Vision Language Models** bachelor thesis.

This project uses the **MTRE framework** with modifications needed for analysis of hyperparameter influence on hallucination detection. 
Any missing content can be generated or retrieved following the `.MTRE_project_experiments.ipynb`.

## MTRE Framework

The original and unchanged MTRE framework as well as steps to use it can be found here: https://github.com/lanl/MTRE/tree/main
